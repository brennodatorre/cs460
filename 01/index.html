RO.ME "3 DREAMS OF BLACK"

RO.ME is a very cool webgl demo, to be honest it was quite interesting to experience what it is capable of It did not feel like I was controlling the "game" in my browser. Putting into contrast to the old flash games that we all grew up using, it is mind blowing that such things can be rendered and run within the bounds of the web browser. No FPS drops was crazy.

Technology: It uses the computer's graphic card to genarate the experience in the browser. This is done with html and javascript.
